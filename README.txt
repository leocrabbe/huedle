Huedle is a simple colour-guessing game with 3 game modes. A randomly chosen colour is always displayed at the top of the page.
In "Name Mode", the user is given 5 hue names to choose from.
In "Colour Mode", the user is asked to select the matching colour out of 15 randomly generated similar pallettes. 
In "Impossible Mode", the user is asked to type the name of the colour in.
